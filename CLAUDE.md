# Lukapp

## ULTRON project name: `lukapp`
Al iniciar → `/start` → elegir "lukapp"

## Stack
- **Frontend:** Next.js 16 + TypeScript (`lukapp/luka-f/`)
- **Backend:** Express + TypeScript + Prisma (`lukapp/luka-b/`)
- **BD:** Supabase + PostgreSQL + Prisma ORM
- **IA:** Anthropic Claude API — coach financiero con voz
- **Auth:** Supabase Auth
